# Руководство по вкладу в проект

## Как помочь проекту

1. **Сообщайте об ошибках**
   - Используйте Issues на GitHub
   - Опишите шаги для воспроизведения
   - Укажите версию Python и ОС

2. **Предлагайте улучшения**
   - Создайте Issue с описанием фичи
   - Обсудите перед реализацией

3. **Отправляйте пул-реквесты**
   - Форкните репозиторий
   - Создайте ветку для ваших изменений
   - Следуйте стилю кода проекта
   - Добавьте тесты (если применимо)

## Стандарты кода

- Используйте Python 3.12+
- Следуйте PEP 8
- Документируйте новые функции
- Добавляйте типизацию (type hints)
